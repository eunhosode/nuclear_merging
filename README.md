# Nuclear Merging

## Gameplay
빨간색 입자: 양성자
파란색 입자: 중성자
초록색 입자: 광자

광자에 맞으면 바로 핵 붕괴, 양성자와 중성자를 먹으며 원자를 성장시키다가 불안정한 핵을 만들면 핵 붕괴할 수 있음
현재 36번 크립톤까지 구현되어있음

## Controls
← → : 이동

## Game Modes
Easy
  일반입자 속도 3, 중성자 붕괴시 소멸
  입자 비율 1:2:0 (양성자:중성자:광자)

Normal
  일반입자 속도 3, 광자 속도 6, 중성자 붕괴시 소멸
  입자 비율 3:6:1

Hard
  일반입자 속도 3, 광자 속도 9, 중성자 붕괴시 소멸
  입자 비율 3:6:1

Speedy
  일반입자 속도 6, 광자 속도 12, 중성자 붕괴시 소멸
  입자 비율 6:12:1

Is it snail?
  일반입자 속도 1, 광자 속도 18, 중성자 붕괴시 소멸
  입자 비율 3:6:1

Choking
  일반입자 속도 3, 광자 속도 1, 중성자 붕괴시 소멸
  입자 비율 1:2:3

Miserable
  일반입자 속도 3, 광자 속도 9, 중성자 붕괴시 소멸
  입자 비율 1:2:3

Schrödinger's particle
  일반입자 속도 3, 광자 속도 12, 양성자 중성자 모두 75% 확률로 붕괴해 광자 형성
  입자 비율 1:1:0

Midgley's Gas
  일반입자 속도 3, 광자 속도 18, 양성자 중성자 모두 75% 확률로 붕괴해 광자 형성
  입자 비율 1:2:3

Permian
  일반입자 속도 18, 광자 속도 18, 양성자 중성자 모두 90% 확률로 붕괴해 광자 형성
  입자 비율 1:1:3

Fog
  일반입자 속도 3, 광자 속도 6, 중성자 붕괴시 소멸
  입자 비율 3:6:1
  맵 중간에 안개가 생겨서 입자가 제대로 안 보임

Paralysis
  일반입자 속도 3, 광자 속도 6, 중성자 붕괴시 소멸
  입자 비율 3:6:1
  10초마다 50% 확률로 3초간 조종 마비가 발생해 가만히 있음
